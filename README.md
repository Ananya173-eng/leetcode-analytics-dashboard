# 🚀 LeetCode Analytics Dashboard

A modern and interactive LeetCode Analytics Dashboard that helps users analyze their coding journey with insightful visualizations, contest statistics, skill analysis, and goal tracking. Built using React.js, Node.js, Express.js, MongoDB, Tailwind CSS, Recharts, and Framer Motion.

---

## 📌 Project Overview

The LeetCode Analytics Dashboard transforms raw coding statistics into meaningful insights through interactive charts and reports. It allows users to monitor their coding consistency, analyze contest performance, track goals, and identify strengths and weak areas for interview preparation.

## ⚙️ Installation

### Clone the Repository

git clone https://github.com/Ananya173-eng/leetcode-analytics-dashboard.git
### Install Dependencies

#### Frontend

cd frontend
npm install
#### Backend

cd backend
npm install
---

## ▶️ Run the Project

### Start Backend

cd backend
npm start
### Start Frontend

cd frontend
npm run dev
